为什么按钮只叫 Open？

因为这个分组会引入一个新问题：用户要先理解“为什么同样是工作单元，有些在 Tasks，有些在 Tmux Sessions”。这不是用户真正关心的。

  • 用户关心的是：我要打开哪个东西。
  • terminal task 没有多窗口能力，所以一层就够。
  • tmux 有多窗口能力，所以只有它展开子窗口。
  • runtime 差异用卡片形态表达，不要让按钮文案变成概念说明。
  • 用户点击时只关心打开这个工作单元；是 terminal 还是 tmux session，应该由系统处理。
  • 不要用内部缩写标签。它们解释了实现，却没有帮助用户完成任务。
