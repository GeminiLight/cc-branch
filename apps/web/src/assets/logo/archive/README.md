# CC Branch Logo 设计思路（第三版）

## 前两版为什么错了

**第一版**在画产品架构图：矩形、连线、圆点。那不是 Logo，是 PPT 插图。

**第二版**在做抽象图形堆砌：萌芽、水滴、丝带。这些东西放在任何 SaaS 上都成立，和 CC Branch 没有灵魂连接。

**两版共同的致命伤**：
1. 紫色渐变 —— 和市面上所有 AI 产品混在一起，没有辨识度
2. 圆形背景 + 中心图形 —— 像 App Store 图标模板，不是品牌符号
3. 从「功能」出发，而不是从「品牌叙事」出发

---

## CC Branch 到底是谁

重新读了一遍项目的 README、architecture 和场景示例，我提炼出三个核心事实：

1. **它不是 AI 产品，是开发者基础设施**
   - 竞品不是 ChatGPT，是「手动开 tmux、记 session ID、反复配环境」的繁琐
   - 它的同类是 Git、Docker、Homebrew —— 不花哨，但好用到离不开

2. **它的核心价值是「秩序」**
   - 用一条命令，把混乱的多 Agent 终端环境变成可重复、可恢复的有序工作空间
   - 配置即代码（YAML），状态可持久，团队可共享

3. **它的核心体验是「终端窗格分割」**
   - 底层是 tmux，用户每天看到的视觉就是：深色背景上的网格线、不同大小的窗格、各个 Agent 在里面跑

**所以品牌气质应该是：秩序、精确、硬核，但有一丝温度。**

## 色彩为什么换了

| 旧方案 | 问题 |
|--------|------|
| 紫色渐变 `#7e14ff` → `#c084fc` | 和所有 AI SaaS 撞车，有廉价「科技感」 |
| 淡紫背景 `#f8f5ff` | 像医疗 App，没有开发者工具的硬核感 |

新色彩系统的逻辑：

| 颜色 | 色值 | 来源 | 情绪 |
|------|------|------|------|
| 深炭灰 | `#1c1917` | 终端背景的黑，但不是纯黑，带一点暖棕 | 专注、深夜、硬核 |
| 暖白 | `#fafaf9` | 终端文字的白，但不是纯白，带一点米 | 清晰、不刺眼、有温度 |
| 赤陶 | `#c2410c` | 陶土、砖墙、暖炉 | 人的介入、生命力、从秩序中浮出的意外 |

深炭灰 + 赤陶是一个在开发者工具里几乎没有撞车的组合。它不像 SaaS，像**一支好钢笔、一个机械键盘、一杯深夜的威士忌**——专业，但有性格。

## 最终方案：The Golden Pane（金格窗）

**文件**: `proposal-final.svg`

### 形态

一个深炭灰色的圆角方块，内部被白色线条分割成三块窗格。

分割比例遵循**黄金比例**（φ ≈ 1.618）：
- 竖线把方块分成 57:35（左宽 : 右宽）
- 横线把右半分成 57:35（上高 : 下高）
- 结果是：左上（高瘦）、右上（扁横）、右下（正方形）

右下角那个最小的正方形，不是乖乖贴着边框的——它**旋转了 45°，变成一颗赤陶色的菱形**，像从网格中微微浮起的一块拼图。

### 为什么是黄金比例

不是炫技。CC Branch 的核心是「用配置精确地定义工作空间」，黄金比例是视觉上的「精确」最自然的表达。三个区域的大小关系是和谐的，不需要解释，眼睛会觉得「对」。

### 为什么右下角要旋转

**网格 = 秩序**。YAML 定义的一切，tmux 精确还原，这是 CC Branch 的确定性。

**旋转的菱形 = 分支与可能性**。从秩序中生长出的新窗口、新 Agent、新的工作状态。它是整个设计里唯一「不守规矩」的元素，也是唯一的彩色元素——因为分支才是这个产品存在的理由。

### 为什么叫「金格窗」

- **金** = 黄金比例，也是赤陶色在深色背景上的光泽感
- **格** = 窗格（pane），tmux 的核心视觉单元
- **窗** = 窗口、workspace、看向工作的视角

## 尺寸适配

| 尺寸 | 呈现方式 |
|------|---------|
| 16×16 favicon | 深炭灰方块 + 右下角赤陶色菱形（线条隐去） |
| 32×32 | 深炭灰方块 + L 形白线 + 菱形 |
| 64×64 以上 | 完整黄金比例网格 + 旋转菱形 |
| 大背景（官网 hero） | 可放大看线条的精确比例和圆角细节 |

---

## 使用场景建议

- **终端/CLI 输出**：16×16 简化版，菱形作为状态指示器
- **GitHub README**：64×64 完整版，配深炭灰或暖白背景
- **Desktop App Icon**：120×120 完整版，圆角外框增加实体感
- **Social Preview / 海报**：放大到 1024×1024，菱形的赤陶色可作为整个画面的 accent 色

## 如果还要调整

这个方案目前只有一个。如果方向对了但细节需要调，可以从这几个维度改：

1. **分割比例**：黄金比例可以调为 1:1（对称）或 2:1（更强烈的对比），气质会不同
2. **旋转角度**：45° 是菱形，30° 是倾斜方块，0° 是规矩方块——"不守规矩"的程度
3. **菱形颜色**：赤陶是温暖，如果品牌要更冷，可以换成青金石蓝 `#1e3a5f` 或氧化铜绿 `#2d5a4a`
4. **外框圆角**：目前 rx=18，调大可更柔和，调小可更硬核
