# V3 方案一览

5 个不同方向，每个配了不同的色彩气质。

| 文件 | 形态 | 配色 | 气质 |
|------|------|------|------|
| `v3-stack.svg` | 两个 pane 错开层叠 | 深墨蓝 + 电光青 | 终端黑客感，活跃窗口浮出 |
| `v3-bracket.svg` | `[ \| ]` 代码符号 | 石板灰 + 珊瑚橙 | 现代温暖，一眼代码 |
| `v3-twin.svg` | 两个 pane 并排错落 | 暖黑 + 赤陶 | 大地色调，沉稳有温度 |
| `v3-split.svg` | 单窗口被粗线分割 | 午夜紫黑 + 荧光绿 | 赛博朋克，高对比吸睛 |
| `v3-tab.svg` | 浏览器标签层叠 | 深蓝 + 纯白 + 青 | 极简干净，最像日常界面 |

**小尺寸表现**：`stack` 和 `tab` 的方块轮廓在 16×16 下最清晰；`bracket` 的 `[ \| ]` 在 32×32 以上才完整；`split` 的荧光绿线是最强的视觉锚点。
