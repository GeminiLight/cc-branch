# V4 方案：三 pane 变体

基于 v3-twin 的反馈：三个方块、大小不一。

| 文件 | 排列方式 | 配色 |
|------|---------|------|
| `v4-wave.svg` | 底部对齐，顶部波浪（低-高-低） | 暖黑 + 赤陶 |
| `v4-city.svg` | 底部对齐，左大右小（天际线） | 暖黑 + 赤陶 + 深灰 |
| `v4-stack.svg` | 向右上方错开层叠（扑克牌） | 暖黑 → 深赤陶 → 亮赤陶 |
| `v4-grid.svg` | 左边大 pane + 右边上下两个小 pane | 暖黑 + 赤陶 + 深灰 |
| `v4-wave-cyan.svg` | 同 wave 形态，换深蓝+电光青配色 | 深墨蓝 + 电光青 |
